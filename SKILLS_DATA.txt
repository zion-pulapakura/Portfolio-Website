SKILLS DATA FILE
================

Edit the skills.json file in src/data/ to add or modify skills for each category.

Current structure:
{
  "aiMl": [
    "Add your AI/ML skills here",
    "One skill per line"
  ],
  "fullstack": [
    "Add your fullstack skills here",
    "One skill per line"
  ],
  "infra": [
    "Add your infrastructure skills here",
    "One skill per line"
  ]
}

INSTRUCTIONS:
1. Open src/data/skills.json
2. Add or modify skills in the appropriate array
3. Each skill should be a string in quotes
4. Skills will automatically appear as pills when hovering over the skill boxes
5. Make sure to keep valid JSON format (commas between items, no trailing comma)

EXAMPLE:
{
  "aiMl": [
    "Python",
    "TensorFlow",
    "PyTorch",
    "Machine Learning",
    "Deep Learning"
  ],
  "fullstack": [
    "React",
    "TypeScript",
    "Node.js",
    "Express",
    "MongoDB"
  ],
  "infra": [
    "AWS",
    "Docker",
    "Kubernetes",
    "CI/CD",
    "Terraform"
  ]
}
